## Adding todos

To store a new todo, use the `todo` command.

Format: `todo [description]`

Example: `todo Buy groceries`

This will add a new todo with the description "Buy groceries" to your list of tasks.

## Adding deadlines

To store a new deadline, use the `deadline` command.

Format: `deadline [description] /by [date]`

Example: `deadline Submit report /by 2026-02-20 2359`

This will add a new deadline with the description "Submit report" and a due date of February 20, 2026 at 11:59 PM.
Note: The date must be in the format `YYYY-MM-DD HHMM` (e.g., `2026-02-20 2359`).

## Adding events

To store a new event, use the `event` command.

Format: `event [description] /from [date] /to [date]`

Example: `event Weekly briefing /from 2026-02-20 1600 /to 2026-02-20 1800`

This will add a new event with the description "Weekly briefing" that starts on February 20, 2026 at 4:00 PM and ends at
6:00 PM.
Note: The date must be in the format `YYYY-MM-DD HHMM` (e.g., `2026-02-20 1600`).

## Viewing tasks

To view all your tasks, use the `list` command.

Format: `list`

This will display a list of all your tasks, including todos, deadlines and events, along with their status (done or not
done).

## Marking task completion status

To mark a task as done, use the `mark` command. Similarly, use the `unmark` command to mark a task as not done.

Format: `mark [task number]` | `unmark [task number]`

Example: `mark 2` | `unmark 2`

This will mark the task with the specified task number as done / not done. You can find the task number by using the
`list` command.

## Deleting tasks

To delete a task, use the `delete` command.

Format: `delete [task number]`

Example: `delete 3`

This will delete the task with the specified task number from your list. You can find the task number by using the
`list` command.

## Searching for tasks

To search for tasks containing a specific keyword, use the `find` command.

Format: `find [keyword]`

Example: `find report`

This will display a list of all tasks that contain the keyword "report" in their description.

## Exiting the application

To exit the application, use the `bye` command.

Format: `bye`

This will exit the application. You can restart the application later to access your saved tasks.
